## Trend

User-facing name for the **umbrella page** that represents a meme, creator, brand, or movement. Multiple **tokens** can exist under one trend. (Engineering and older UI may still say **IP**.)

## Token

A **single tradable market** launched under a trend. Sometimes labeled **Remix** in UI or code.

## Remix

Legacy label for **token** in parts of the app—same object.

## Creator trend

Owner-led umbrella. Token launch optional at create time. Verified through **owner claim** flows.

## Community trend

Holder-coordinated umbrella. Token launch **required** at create time. Verified through **milestones**.

## Verification

Process that upgrades an umbrella from unverified to verified so economics and governance can safely attach.

## ITL (IP Token License)

Legal framework binding **verified** umbrellas—read the PDF linked from [IP Token License (ITL)](/user-guides/verification/ip-token-license-itl).

## IPA (IP Account)

Story **IP Account** address used in advanced claim paths.

## Endorsement

Signal from a verified **creator trend owner** on a token (**Official**, **Liked**, **Flagged**).

## Anti-snipe

Launch option limiting early wallet concentration—see [Metadata, socials, and anti-snipe](/user-guides/launch/metadata-socials-and-antisnipe).

## Canonical voting token

For **community** governance, the **first token** created under the umbrella gates proposal/vote eligibility.

## Governance pass rule

- **YES > NO**
- **YES ≥ REQUIRED_YES** with **REQUIRED_YES = 0.10 + 1 × NO** (10% baseline + opposition scaling)

## Points

App-layer ledger for quests, referrals (partial), trading seasons, etc.—see [Points and leaderboard](/user-guides/rewards/points-and-leaderboard).

## Referral link

`https://ip.world?ref=YOURCODE` — see [Referral program](/user-guides/rewards/referrals).

## UGC

Social posting programs that route fee share to creators—see [UGC program](/user-guides/rewards/ugc-program).

## Holder rewards

Windowed payouts to scored holders—see [Holder rewards](/user-guides/rewards/holder-rewards).

## Relay

Cross-chain liquidity movement (e.g., Story **IP** to Solana **SOL**) via Relay quotes—see [Payment methods](/user-guides/trading/payment-methods).

## Embedded wallet

MPC wallet from Dynamic—Story EVM identity + Solana funding wallet; see [Embedded wallets](/user-guides/account/embedded-wallets).
