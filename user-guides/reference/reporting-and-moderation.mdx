## What you can report

`ip.world` supports in-app reporting for several things, including:

- **IPs**
- **Tokens**
- **Feeds** (X / TikTok posts shown in Feeds)
- **Comments**
- **Takes**

Reporting is meant to surface issues to the moderation system.

## Where the Report button is

You'll see **Report** entrypoints in places like:

- token pages
- IP pages (including feed cards)
- comment items

## Common report reasons (what you'll see in the UI)

The exact options depend on what you're reporting, but common reasons include:

- **Inappropriate content**
- **Spam**
- **Duplicate IP**
- **Wrong owner** (for creator attribution / ownership signals)
- **Wrong IP** (token appears linked to the wrong IP)
- **IP type incorrect** / **wrong type** (when applicable)

## Duplicate IP reports

If an IP looks like a duplicate of an existing IP, use the **Duplicate IP** report option and select the canonical IP it should be merged into.
