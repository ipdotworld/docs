## Should I create a new IP or a token?

If the IP already exists and it's basically the same thing, create a **token under the existing IP**.

Start here: [IPs & tokens](/user-guides/ips-and-tokens)

## Why does ip.world care about verification?

Because identity signals matter when there are many tokens around the same meme/creator. Verification reduces confusion and enables owner features.

Start here: [Verification overview](/user-guides/verification/verification-overview)

## What does "READY FOR CLAIM" mean?

It means the creator IP has been approved and the owner (matching connected social identity) can claim it.

Start here: [Claiming a creator IP](/user-guides/verification/claiming-a-creator-ip)

## What is anti-snipe?

A launch option that limits post-launch wallet concentration for a short window.

See: [Metadata, socials, & anti-snipe](/user-guides/create/metadata-socials-and-antisnipe)
