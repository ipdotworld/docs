## Holders tab

Shows **top holders** with addresses (and usernames when resolved). Empty state copy in-app: **There aren't any holders right now** when metrics have not populated.

![Holders tab figure](/images/token-holders.svg)

## Transactions tab

Lists swaps relevant to the token; the table is backed by the same swap feeds used for PnL helpers in the app (`swap-table`, `use-swaps-pnl`).

![Transactions tab figure](/images/token-transactions.svg)
