## Tabs you should know

On desktop, the token page uses a tab strip (see `token-data.tsx` in the app) for:

- **Feed** — social posts for this token.
- **Holders** — top holders list.
- **Transactions** — swap history table.
- **Comments** — threaded discussion.
- **Treasury** — proposals when governance is active for that context.

![Token tab strip figure](/images/token-tabs-desktop.svg)

## How it works

The lavender **How it works** button opens an in-app explainer modal for economics context—use it when you are unsure where fees flow.

## Mobile

On small screens, **Trade** and **Comments** can open as **bottom sheets** so you do not lose chart context—swipe or tap outside to dismiss per mobile patterns in the token layout.
