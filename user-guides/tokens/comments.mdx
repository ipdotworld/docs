Every token has a **Comments** tab for threaded discussion between traders, holders, and creators.

## Posting

You need to be signed in with an active wallet to comment. If you are not connected, the composer shows:

> **Please connect your wallet before submitting a comment**

Sign in and try again.

## Replies

Click any comment to open its thread and post a reply. The UI shows your message immediately (optimistic), then reconciles with server state once the write lands.

![Comments tab figure](/images/token-comments.svg)

## Empty state

**No comments yet** shows until the first comment is posted.
