## The 10-second version

`ip.world` turns **Internet IP** (a meme, a creator, a brand, a trend) into a **home** where:

- The IP has a canonical identity (name, image, socials, verification state).
- Multiple **tokens** can exist “under” that IP (think: markets/remixes around the same idea).
- Ownership, verification, and governance tools live next to the market.

If you’re a degen: it’s “meme markets with structure.”
If you’re a normie: it’s “a page for a thing, with official-ish links, and tokens attached to it.”

## Two objects: IPs and tokens

- **IP**: the umbrella identity. One IP can have many tokens under it.
- **Token**: a specific market/remix created under an IP.

Read next: [IPs & tokens](/user-guides/ips-and-tokens).
