## The mental map

Most user flows on `ip.world` live in a small set of places:

- **Explore**: discover IPs and tokens
- **IP pages**: the “home” for a meme/creator/brand/trend (with tokens listed under it)
- **Token pages**: the market for a specific token under an IP
- **Create**: create an IP or create a token under an IP
- **Profile**: your stats and referral link
- **Settings**: username/bio/avatar, connect socials, verify TikTok

## Explore

Use Explore when you’re trying to answer:

- “What’s happening right now?”
- “Where are the tokens under this IP?”
- “Does this IP already exist?”
