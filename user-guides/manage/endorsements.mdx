## What an endorsement is

An endorsement is a **signal from a verified IP owner** applied to a token under their IP.

It does not magically guarantee price performance — it's an identity signal.

## Endorsement types

`ip.world` currently shows these owner endorsement states:

- **Official Token**
- **Liked By Owner**
- **Flagged by Owner**

You'll see these as a visible indicator on token cards and token pages.

## How to use endorsements as a user

- **Official Token**: the owner is explicitly signaling "this is the canonical/official token for this IP."
- **Liked By Owner**: the owner is positively acknowledging the token.
- **Flagged by Owner**: the owner is signaling the token is not endorsed and may be misleading.
