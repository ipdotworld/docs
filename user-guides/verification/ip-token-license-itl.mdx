## What is the ITL?

The **IP Token License (ITL)** is the licensing framework used by `ip.world` for verified IP ownership.

In the product UI, you’ll see copy like:

> “Once verified, an IP will be bound to the IP Token License (ITL).”

## Why it matters

ITL is designed to make it clearer what it means to:

- be a verified IP owner on `ip.world`
- have tokens and markets created under your IP identity

## Where to read the license

The current ITL document is hosted here:

`https://github.com/ipdotworld/protocol/blob/main/ip-token-license-v1.0.pdf`
