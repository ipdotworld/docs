## Your referral code and link

Your profile includes a **referral code** and a shareable link in this format:

`https://ip.world?ref=YOUR_CODE`

## Where to find it

- Open your **Profile**
- Copy the referral code or referral link from the profile UI

<Note>
  **Screenshot**: Add a screenshot of the profile header showing the referral code + copy button.
</Note>

## Tracking referrals

If you have referred users, you may see:

- a referrals table in your profile
- "points you earned" attributed to referred users

See also: [Rewards & points](/user-guides/rewards/rewards-and-points)
