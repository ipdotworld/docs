## Before you create

<Warning>
  If a trend already exists for the idea you have in mind, do **not** create a new one. Launch a token under the existing trend instead.
</Warning>

As you type a trend name, ip.world searches for fuzzy matches and surfaces existing trends in a dropdown. If your trend is a duplicate of an existing one, moderators will merge it later and move any tokens you created under the canonical trend.

![Create trend figure](/images/create-trend.svg)

## Ownership types

ip.world has two ownership models for trends:

- **Creator trend** - the trend has a clear real-world owner (a person, a brand, a project). The owner links an X or TikTok account as a verification anchor. Launching a token at creation time is optional.
- **Community trend** - a bottom-up movement owned collectively by its holders. Launching a token at creation time is required.

## Steps

<Steps>
  <Step title="Open Create">
    From the nav, click **Create**.
  </Step>
  <Step title="Enter trend identity">
    Add an image, the canonical name, and aliases. Aliases help search catch alternate spellings and prevent duplicate trends.
  </Step>
  <Step title="Pick a category">
    Pick the category that best describes the trend: Pet, Person, Trend, Game, Technology, Meme, Movement, Brand, Character, Art, or Misc. The category also gates which verification tracks are available later.
  </Step>
  <Step title="Pick ownership type">
    Choose **Creator** or **Community**. Creator trends can launch without a token; community trends require a token at creation.
  </Step>
  <Step title="Add description and socials">
    Fill in a description and the relevant social links (X, TikTok, Instagram, Telegram, website). Traders use these to judge whether the trend is legitimate.
  </Step>
  <Step title="Confirm">
    Submit. If you are launching a token in the same flow, complete the wallet prompt.
  </Step>
</Steps>

## Launch fees

The launch fee is **30 IP**, charged on top of the initial buy amount you select. The fee exists to deter spam and bot-launched trends.

<Note>
  The launch fee is set by the protocol and can change. The exact amount is always shown in the wallet prompt before you sign - read it before confirming.
</Note>
