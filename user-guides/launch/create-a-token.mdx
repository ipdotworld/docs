## When to use this flow

Use **Launch token** when the **trend already exists** and you are adding another market under it.

See [Trends and tokens](/user-guides/trends-and-tokens).

![Create token figure](/images/create-token.svg)

## Steps

<Steps>
  <Step title="Open Create token">
    **Create → Create token** (may read “remix” in UI).
  </Step>
  <Step title="Pick the trend">
    Search and select the umbrella you are launching under.
  </Step>
  <Step title="Token identity">
    **Image**, **name**, **ticker** (`$TICKER`), **description**, **socials** (token socials include **Telegram**).
  </Step>
  <Step title="Anti-snipe (optional)">
    Toggle early concentration limits—[details](/user-guides/launch/metadata-socials-and-antisnipe).
  </Step>
  <Step title="Creator buy">
    Choose buy size in **`IP`**; the UI shows estimated tokens and % of supply.
  </Step>
  <Step title="Launch">
    Submit and confirm in your wallet.
  </Step>
</Steps>
