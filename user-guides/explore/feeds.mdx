## What "Feeds" are

On an IP page, **Feeds** is where you browse social posts associated with that IP.

You'll typically see:

- X (Twitter) embeds
- TikTok embeds

## Filters

On desktop, Feeds supports filtering by source:

- All
- Twitter
- TikTok

## Sorting

Feeds supports sorting modes like:

- Newest / Oldest
- Most Featured / Least Featured

## Refresh

There is a refresh button to pull the latest feed state.
