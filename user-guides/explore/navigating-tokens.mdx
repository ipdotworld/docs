## What a token page is

A token page is a specific market created **under an IP**.

It typically includes:

- token identity (name, ticker, image, description)
- socials attached to the token
- market information and trading UI
- endorsement signals (when applicable)

## Endorsement signals (when applicable)

If the IP owner is verified, tokens can show an endorsement indicator:

- **Official Token**
- **Liked By Owner**
- **Flagged by Owner**

See: [Endorsements](/user-guides/manage/endorsements)

## Anti-snipe messaging

Some tokens are launched with anti-snipe protection. If you see errors like "Anti-snipe limit exceeded," it means the trade would violate the post-launch holding limits.

See: [Metadata, socials, & anti-snipe](/user-guides/create/metadata-socials-and-antisnipe)
