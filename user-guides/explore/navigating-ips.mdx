## What an IP page is

An IP page is the “home” for an identity (a meme, creator, brand, or trend).

It typically includes:

- IP identity (name, image, aliases)
- socials (if provided)
- verification/ownership status
- the list of tokens created under it

## Ownership & verification panel (important)

Creator IPs have an **IP Ownership** panel with states like:

- **IDENTIFYING IP OWNER**
- **READY FOR CLAIM** (when approved and eligible)
- **VERIFIED / CLAIMED** (when ownership is established)

Community IPs use a different model and can show community verification status.

## Tokens under an IP

Scroll the IP page to see all tokens created under it.

For creator IPs, once the owner is verified they can apply token endorsements:

- Official Token
- Liked By Owner
- Flagged by Owner

See: [Endorsements](/user-guides/manage/endorsements)
