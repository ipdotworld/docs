## Search first, create second

Before you create anything:

1. Search for the IP by name.
2. Check for obvious aliases.
3. Open the IP page and look at tokens already created under it.

<Tip>
  If an IP already exists and it's basically the same idea, create a **token under the existing IP** instead of making a duplicate IP.
</Tip>

## Filtering mindset

When you filter/sort, you're usually answering one of these:

- “What's trending right now?”
- “What has real holders/liquidity?”
- “What's a fresh launch vs a mature market?”
