## Why connect X

Linking X helps the app attribute posts for **UGC** eligibility, shows your handle on profile when enabled, and supports verification flows that compare owner accounts.

## Steps

<Steps>
  <Step title="Open Settings">
    Go to **Settings** in the app.
  </Step>
  <Step title="Start X linking">
    Use the **Connect X** (or equivalent) control shown in the socials section.
  </Step>
  <Step title="Approve OAuth">
    Complete the X authorization in the popup window.
  </Step>
  <Step title="Confirm status">
    Return to Settings and confirm the connected badge or timestamp shown in UI.
  </Step>
</Steps>

## Disconnect

Use the disconnect / unlink control in Settings if you need to rotate accounts.
