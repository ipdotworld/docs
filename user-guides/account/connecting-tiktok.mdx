## Why TikTok is different

TikTok linking for crypto apps often cannot rely on a normal consumer OAuth flow. ip.world uses a **one-time code** plus **Telegram bot** validation pattern (see product notes in the repo’s Notion export: **User Profile**).

## Steps

<Steps>
  <Step title="Open Settings → TikTok">
    Start the TikTok verification flow from Settings.
  </Step>
  <Step title="Submit your handle">
    Enter the TikTok username you control.
  </Step>
  <Step title="Copy the one-time code">
    The app displays a code generated by the backend.
  </Step>
  <Step title="Message the Telegram bot">
    Open the official ip.world Telegram bot and send the code as instructed in-app.
  </Step>
  <Step title="Wait for confirmation">
    When `/validate-code` succeeds, Settings should show TikTok as linked.
  </Step>
</Steps>

<Warning>
  Treat the one-time code like a password: anyone with the code could bind your TikTok slot until it expires or is consumed.
</Warning>
