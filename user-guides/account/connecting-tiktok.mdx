TikTok does not provide a consumer OAuth flow suitable for ip.world, so verification runs through a one-time code sent via the ip.world Telegram bot. The code proves you control the TikTok handle you claim.

## Steps

<Steps>
  <Step title="Open Settings → TikTok">
    Start the TikTok verification flow from Settings.
  </Step>
  <Step title="Submit your handle">
    Enter the TikTok username you own.
  </Step>
  <Step title="Copy the one-time code">
    The app displays a short-lived code. Copy it.
  </Step>
  <Step title="Send the code to the Telegram bot">
    Open the ip.world Telegram bot linked in the UI and send the code as instructed.
  </Step>
  <Step title="Confirm linked">
    Once the bot confirms the code, Settings updates to show TikTok as linked.
  </Step>
</Steps>

<Warning>
  Treat the one-time code like a password. Anyone with the code before it expires can bind your TikTok handle to their account.
</Warning>
