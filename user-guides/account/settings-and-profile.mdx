## What you can edit

Use **Settings** to update the fields the app exposes there—typically **username**, **bio**, and **avatar** upload. Exact field availability follows the live Settings page.

## Security Settings

Open **Security Settings** in the Settings sidebar to manage embedded wallets: **Story Wallet** and **Solana Wallet** addresses, plus **Show Recovery Key** to export each embedded wallet’s private key through Dynamic’s secure modal. Step-by-step: [Embedded wallets](/user-guides/account/embedded-wallets#export-your-embedded-private-key-settings).
