## When you should create a token (not a new IP)

Create a token when the IP already exists and you're launching a specific market/remix under it.

See: [IPs & tokens](/user-guides/ips-and-tokens)

## Create flow (under an existing IP)

<Steps>
  <Step title="Open Create Token">
    In the app, go to Create → **Create Token** (remix an existing IP).
  </Step>
  <Step title="Pick the IP">
    Search for the IP and open the token creation flow for that IP.
  </Step>
  <Step title="Fill token details">
    - **Image**
    - **Token Name**
    - **Ticker** (e.g. `$TICKER`)
    - **Description**
    - **Socials** (token socials include Telegram)
  </Step>
  <Step title="Choose anti-snipe (optional)">
    You can enable anti-snipe protection at launch.
  </Step>
  <Step title="Choose buy amount">
    You specify a buy amount (in `IP`) and the UI shows the estimated tokens and % of supply.
  </Step>
  <Step title="Launch">
    Submit and confirm the transaction in your wallet.
  </Step>
</Steps>
