## Your link

`https://ip.world?ref=YOURCODE`

The `ref` parameter maps to your public username / code—keep it stable so attribution does not break.

![Referral profile figure](/images/profile-referrals.svg)

## New user referral

A successful referral that creates a new user awards **100 points** to the referrer (engineering default in `REWARDS_AND_REFERRAL_SYSTEM.md`).

## Referral bonus on downstream points

Referrers can earn **10%** of many downstream point events that flow through the main reward service—see [Points and leaderboard](/user-guides/rewards/points-and-leaderboard).

## Tracking

Profile surfaces tables summarizing referred users and points earned where the UI enables it.
