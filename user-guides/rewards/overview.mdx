## Read this once

ip.world runs several **parallel** incentive systems:

| Program | What it is | Where you see it |
|---------|------------|------------------|
| **Points** | On-platform achievements + trading season volume + quests | **Leaderboard**, profile |
| **Referrals** | Invite links that attribute signups / volume | **Profile**, search dialog |
| **UGC payouts** | Social creators earning from fee pools | Token analytics / UGC surfaces |
| **Holder rewards** | Longer-term holder windows from shared pools | Token UI / dashboards |
| **Owner / treasury fees** | Verified owner or community treasury routes | Owner dashboard, proposals |

![Earning systems diagram](/images/diagram-earning-flow.svg)

## Disambiguation (please read)

- **Trend owner economics** ≠ **UGC creator program**. Owners earn through verified fee routes; UGC creators earn by qualifying social posts that drive attention and volume.
- **Points** are not the same as **on-chain reward claims**—points feed leaderboards and future programs as announced in-product.
