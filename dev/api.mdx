## Notes

- Paths are relative to the deployment host (`https://ip.world`).
- Most routes **proxy** or enrich data from `NEXT_PUBLIC_API_BASE_URL` (`/api/v1` on the Go service).
- **Auth**: send `X-Authorization: Bearer <jwt>` when the route uses `fetchJSONWithAuth` patterns.

## Users & search

| Method | Path | Purpose |
| --- | --- | --- |
| `GET` | `/api/users/current` | Session user profile |
| `POST` | `/api/users` | Create/update user payloads from client actions |
| `GET` | `/api/users/search?q=` | Search leaderboard users |
| `GET` | `/api/users/referral` | Referral metadata |
