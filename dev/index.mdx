## Audience

- **Integrators** building bots or internal tools that reuse the **Next.js BFF** routes under `https://ip.world/api/*`.
- **Backend engineers** extending the Go API hosted at `NEXT_PUBLIC_API_BASE_URL` (for example `https://dev-api.ip.world/api/v1` in tests).

This documentation **does not** expose private admin dashboards or internal-only operator APIs.
